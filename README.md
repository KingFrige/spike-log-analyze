# README

## requirements

centos
-----------

```bash
$ sudo yum install python3-tkinter
$ pip3 install -r requirements.txt --user
```

debian/ubuntu
---------------

```bash
$ sudo apt-get install python3-tk
$ pip3 install -r requirements.txt --user
```


## run

```
$ python3 plot-histgram.py

# show
$ gio open output/insn-histgram.png
```
